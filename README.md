我是高佳煒，大學就讀於嘉義大學經濟系，碩士畢業於輔仁大學金融與國際企業學系。具財金與資料分析背景，熟悉 Python、計量工具與 AI 工具整合。曾參與塭仔圳輔大醫院財務分析計畫、金融機器學習營、TEJ 交易策略競賽、期貨選擇權期刊投稿與金融科技相關競賽，目前正參與經濟部 AI 新秀培訓專班，強化 AI 與研究整合應用。
